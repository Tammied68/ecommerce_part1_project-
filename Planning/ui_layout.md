# User Interface Layout – eCommerce Application

## 1. Main Screens

### Home Page
- Header with logo, navigation, login/register
- Featured products grid
- Footer with contact info and links

### Product Listing Page
- Grid of products with images, names, and prices
- Filters (category, price, size)
- Pagination

### Product Detail Page
- Large product image
- Description, price, sizes, colors
- Upload‑your‑photo button for custom shirts
- Add to cart button
- Customer reviews section

### Upload Custom Image Page
- File upload field
- Preview of uploaded image
- Validation messages (file type, size)
- Add to cart button

### Cart Page
- List of items with thumbnails
- Quantity selector
- Remove item button
- Order summary (subtotal, tax, total)
- Checkout button

### Checkout Page
- Shipping information form
- Payment method selection
- Order review
- Place order button

### Login / Register Pages
- Email + password fields
- Password reset link
- Error messages for invalid credentials

### Password Reset Pages
- Request reset token
- Enter new password
- Confirmation message

### User Profile Page
- Order history
- Account details
- Option to update information

### Admin Dashboard
- Product management table
- Order management table
- Review moderation
- User permissions
